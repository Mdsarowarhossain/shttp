# shttp
semple http call

##To get a Git project into your build:

Step 1. Add the JitPack repository to your build file


`	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
`



##Step 2. Add the dependency
  `  implementation 'com.github.Mdsarowarhossain:shttp:1.0.0'`
